# Projet Medicare - JEE (Spring Boot)

## Description
Application de gestion hospitalière permettant de gérer les médecins, les patients et les consultations avec export PDF.

## Installation
1. Importer la base de données : le fichier `medicare_db.sql` se trouve à la racine.
2. Configurer le login/password MySQL dans `src/main/resources/application.properties`.
3. Lancer l'application via Maven : `mvn spring-boot:run`.
